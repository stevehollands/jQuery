jQuery : Exercise 7

What is jQuery?

jQuery is a library JavaScript free and cross-platform platform created to facilitate the writing of client-side scripts in the code HTML web pages.

The first version was launched in January 2006 by John Resig.

What can I do with jQuery?

The library contains the following features in particular:

  • Course and modification of the DOM (including selector support CSS 1 to 3 and a basic support of XPath);
  • Events ;
  • Visual effects and animations ;
  • Manipulations of cascading style sheets (adding/deleting classes, attributes...) ;
  • AJAX ;
  • Plugins ;
  • Utilities ( web browser,...).
